UI/UX & Product Design

Design intuitive, engaging digital experiences for users

Course Overview

This programme equips you to design human-centred products. You’ll learn user research techniques, information architecture, interaction design, visual design principles, prototyping in Figma/Adobe XD, accessibility guidelines and product thinking. The course emphasises iterative design and usability testing.

User Research

Conduct interviews, surveys and create personas to understand user needs.

Design Systems

Build component libraries, manage typography and ensure consistent styling.

Prototyping

Create interactive prototypes using Figma or Adobe XD and gather feedback.

Curriculum

User Research & Personas

Interview techniques, surveys, empathy mapping and persona creation.

Information Architecture

Create sitemaps, user flows and organise content for clarity and efficiency.

Visual Design

Learn typography, colour theory, layout and design systems.

Prototyping & Tools

Use Figma/Adobe XD to create wireframes, mockups and interactive prototypes.

Usability Testing

Plan and run usability tests, analyse feedback and iterate designs.
