Cloud & DevOps

Master cloud infrastructure and automate delivery pipelines

Course Overview

Gain expertise in cloud architectures and DevOps practices. You’ll learn AWS and Azure services, containerisation with Docker, orchestration with Kubernetes, CI/CD pipelines, infrastructure as code and monitoring. By the end, you’ll be able to design, deploy and maintain scalable cloud environments.

Cloud Fluency

Understand core services from AWS, Azure and how to choose the right tool for the job.

Containers & Orchestration

Package applications with Docker and manage clusters using Kubernetes.

Automation

Implement CI/CD pipelines, automate deployments and embrace infrastructure as code.

Curriculum

Cloud Fundamentals

Core services of AWS, Azure & GCP, IAM, networking and pricing models.

Containerisation

Build and manage Docker images and compose multi-container applications.

Kubernetes & Orchestration

Deploy, scale and update applications on Kubernetes clusters.

CI/CD & Automation

Set up pipelines with Jenkins or GitHub Actions, and automate testing and deployment.

Infrastructure as Code

Write declarative templates with Terraform and CloudFormation.

Monitoring & Logging

Use Prometheus, Grafana and ELK stack to observe system health and performance.
