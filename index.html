Agentic AI — 30-Day Free Course (Beginner → Builder)

Outcome: Build and deploy your own tool-using LLM agent with planning, memory, and a small capstone.

  1. Days 1–3: Foundations — What is an agent; LLM basics; set up Python; APIs & JSON; a tiny script calling an API.
  2. Days 4–6: Prompting & Tool Use — Prompts, function/tool calling; ReAct loop; build a calculator + web search tool.
  3. Days 7–9: Memory — Short- vs long-term memory; embeddings; Chroma setup; agent with doc retrieval.
  4. Days 10–12: Planning — ReAct, Reflexion, basic task decomposition; research-and-summarize agent.
  5. Days 13–15: Integrations — Read/write files; call 3rd-party APIs (news/Slack/Notion); schedule tasks.
  6. Days 16–18: Reliability — Guardrails, timeouts/retries, evals; logging + simple metrics.
  7. Days 19–21: Multi-Agent — Coordinator + worker pattern; handoffs; failure recovery.
  8. Days 22–24: Performance & Scale — Async, batching, streaming; caching; persisting memory.
  9. Days 25–27: UI & Deploy — CLI → FastAPI/Streamlit; Docker; deploy to a small cloud host.
  10. Days 28–30: Capstone — Ship a working agent (e.g., research → draft → revise). Write a short README & demo.

Prereqs: None. We teach the Python you need as you go.

Generative AI for Product Managers — 8-Week Syllabus

Outcome: Launch AI-powered products with strategic clarity, technical depth, and governance alignment.

  1. Week 1: Strategic Product Leadership in GenAI — vision, RICE/OKRs, market opportunity.
  2. Week 2: Architecture & Technical Foundations — RAG, agentic systems, transformers, compute sizing.
  3. Week 3: Data & Retrieval Pipeline — ingestion, chunking, embeddings, vector DB trade-offs.
  4. Week 4: Agentic Workflow & Multi-Agent Design — orchestration, routers, fallbacks, recovery.
  5. Week 5: MLOps, Deployment & Monitoring — CI/CD, drift/latency/hallucination metrics, rollback.
  6. Week 6: Ethics, Governance & Reliability — bias mitigation, explainability, compliance, auditability.
  7. Week 7: Execution & Stakeholder Alignment — exec storytelling, trade-offs, cross-functional handoffs.
  8. Week 8: Capstone — end-to-end system design, ADRs, final presentation.
