Agentic AI — 30-Day Free Course (Beginner → Builder)

Outcome: Build and deploy your own tool‑using LLM agent with planning, memory, and a small capstone.

  1. Days 1–3: Foundations — what is an agent; LLM basics; Python setup; APIs & JSON.
  2. Days 4–6: Prompting & Tool Use — function calling; ReAct loop; calculator + web search tool.
  3. Days 7–9: Memory — embeddings; vector store; document retrieval.
  4. Days 10–12: Planning — ReAct / Reflexion; task decomposition; research summarizer agent.
  5. Days 13–15: Integrations — read/write files; 3rd‑party APIs (news/Slack/Notion); scheduling.
  6. Days 16–18: Reliability — guardrails; retries/timeouts; logging; quick evals.
  7. Days 19–21: Multi‑Agent — coordinator/worker; handoffs; recovery.
  8. Days 22–24: Performance — async/streaming; caching; persisting memory.
  9. Days 25–27: UI & Deploy — FastAPI/Streamlit; Docker; deploy to a small cloud host.
  10. Days 28–30: Capstone — ship a working agent; README + demo.

Prereqs: None — we teach the Python you need as you go.

Generative AI for Product Managers — 8-Week Syllabus

Outcome: Launch AI-powered products with strategic clarity, technical depth, and governance alignment.

  1. Week 1: Strategic Product Leadership in GenAI — vision, RICE/OKRs, market opportunity.
  2. Week 2: Architecture & Technical Foundations — RAG, agentic systems, transformers, compute sizing.
  3. Week 3: Data & Retrieval Pipeline — ingestion, chunking, embeddings, vector DB trade-offs.
  4. Week 4: Agentic Workflow & Multi-Agent Design — orchestration, routers, fallbacks, recovery.
  5. Week 5: MLOps, Deployment & Monitoring — CI/CD, drift/latency/hallucination metrics, rollback.
  6. Week 6: Ethics, Governance & Reliability — bias mitigation, explainability, compliance, auditability.
  7. Week 7: Execution & Stakeholder Alignment — exec storytelling, trade-offs, cross-functional handoffs.
  8. Week 8: Capstone — end-to-end system design, ADRs, final presentation.

SAP PP–PM–QM — 8-Week Syllabus

Outcome: Deliver integrated Production Planning, Quality Management, and Plant Maintenance in SAP S/4HANA with hands-on confidence and governance alignment.

  1. Week 1: S/4HANA Manufacturing & Asset Management Overview — org structure (plants, storage locations), master-data map, roles, and Fiori workspace.
  2. Week 2: PP Master Data — materials, BOMs, work centers/resources, routings/task lists, production versions; change control & release strategy.
  3. Week 3: Demand & MRP — planning strategies (MTS/MTO), PIRs, MRP Live, exception messages, pegging, lot sizing.
  4. Week 4: Shop-Floor Execution & Capacity — production orders, confirmations, GI/GR, variance & settlement, basic capacity planning and PP/DS overview.
  5. Week 5: QM Foundations & Procurement — MICs, inspection plans & methods, sampling, GR inspections, usage decisions, defect recording.
  6. Week 6: QM in Production & Sales — in-process (03) & GR from production (04) lots, nonconformance handling/CAPA, quality certificates.
  7. Week 7: PM/EAM — technical objects (functional locations, equipment), notifications & orders, preventive maintenance plans, calibration (QM integration), resource scheduling/mobile.
  8. Week 8: Capstone — end-to-end scenario: forecast → MRP → production → inspections → maintenance order → analytics; ADRs and final presentation.
