Data Science & AI

Our Data Science & AI programme provides a comprehensive foundation in data analysis, machine learning and deep learning. Learn Python, statistics, data visualisation and work with popular libraries such as pandas, NumPy, scikit-learn, TensorFlow and Keras. Throughout the course you’ll build real-world projects and explore ethical considerations in AI.

Curriculum

  • Data Science Foundations: statistics, probability, Python and Jupyter notebooks.
  • Data Wrangling & Visualisation: use pandas, NumPy and matplotlib/seaborn for exploratory analysis.
  • Machine Learning: supervised & unsupervised algorithms, model evaluation and hyperparameter tuning.
  • Deep Learning: neural networks, CNNs and RNNs implemented using TensorFlow and Keras.
  • Capstone Project: apply your skills to a real-world problem, from data collection to deployment.
×

Front-End Development

This programme teaches you to craft engaging web experiences. Master HTML5 semantics, modern CSS (Flexbox & Grid), JavaScript ES6+ and responsive design principles. We also explore accessibility guidelines and frameworks like React and Angular so you can build single-page applications.

Curriculum

  • HTML & CSS Fundamentals: semantic markup, selectors, Flexbox, Grid and responsive layouts.
  • JavaScript Basics: variables, functions, objects, ES6 features and DOM manipulation.
  • Frameworks & Tooling: React fundamentals, component lifecycle and an introduction to Angular.
  • Responsive & Accessible Design: mobile-first approaches, ARIA roles and usability testing.
  • Build Tools & Project: version control with Git, npm scripts, webpack and a final portfolio project.
×

Back-End & APIs

Learn server-side programming, database management and API design. This course covers Node.js with Express, Python frameworks like Flask and Django, relational and NoSQL databases, RESTful and GraphQL APIs, authentication and security, microservices and deployment strategies.

Curriculum

  • Server-Side Fundamentals: asynchronous programming patterns with Node.js and Express.
  • Database Systems: design schemas, write queries in PostgreSQL and model data with MongoDB.
  • API Design: build RESTful services, explore GraphQL and document your endpoints.
  • Authentication & Security: implement JWT/OAuth, validation and protect against vulnerabilities.
  • Deployment & Scaling: deploy to cloud platforms, containerise with Docker and scale microservices.
×

Cloud & DevOps

Gain expertise in cloud architectures and DevOps practices. Learn about AWS and Azure services, containerisation with Docker, orchestration with Kubernetes, CI/CD pipelines, infrastructure as code and monitoring. By the end, you’ll design, deploy and maintain scalable cloud environments.

Curriculum

  • Cloud Fundamentals: core services from AWS, Azure and GCP; IAM, networking and pricing models.
  • Containerisation: build and manage Docker images and compose multi-container applications.
  • Kubernetes & Orchestration: deploy, scale and update applications on Kubernetes clusters.
  • CI/CD & Automation: set up pipelines with Jenkins or GitHub Actions, automate testing and deployment.
  • Infrastructure as Code: write declarative templates with Terraform and CloudFormation.
  • Monitoring & Logging: use Prometheus, Grafana and the ELK stack to observe system health.
×

Software Testing & QA

Understand testing methodologies across the software development lifecycle. This course teaches manual testing, test planning, automation frameworks like Selenium and Cypress, unit testing with JUnit/PyTest, as well as performance and security testing. You’ll practise creating test suites, reporting bugs and integrating tests into CI/CD pipelines.

Curriculum

  • Testing Principles: types of testing, SDLC integration and writing effective test plans and cases.
  • Manual & Exploratory Testing: defect tracking, exploratory techniques and test management tools.
  • Automation Frameworks: set up automated tests with Selenium, Cypress and integrate with CI/CD.
  • Unit & Integration Testing: write tests in JUnit and PyTest, mock dependencies and measure coverage.
  • Performance & Security: learn load testing, profiling, vulnerability scanning and secure coding practices.
×

UI/UX & Product Design

This programme equips you to design human-centred products. Learn user research techniques, information architecture, interaction design, visual design principles, prototyping in Figma/Adobe XD, accessibility guidelines and product thinking. The course emphasises iterative design and usability testing.

Curriculum

  • User Research & Personas: interview techniques, surveys, empathy mapping and persona creation.
  • Information Architecture: create sitemaps, user flows and organise content for clarity.
  • Visual Design: typography, colour theory, layout and design systems.
  • Prototyping & Tools: use Figma/Adobe XD to create wireframes, mockups and interactive prototypes.
  • Usability Testing: plan and run usability tests, analyse feedback and iterate on designs.
